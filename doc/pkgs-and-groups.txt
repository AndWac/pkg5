
pkg
Packages and Groups

  It would be nice to have a package FMRI, which would be something like

  pkg://repo/pkg_name@version

  This namespace might be populated with "manifest" and other metadata
  endpoints, as well as the SHA-1 names of the package's included files.

  Packages need a limited set of operations on individual files to
  manipulate the configuration.
